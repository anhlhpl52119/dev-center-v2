---
title: CSS instruction
---

- This project using Tailwind for styling.
- Only use the color token that defined in `./variables.css`

**Example 1:** use background color base `--color-base-abg`

```vue
<div class="bg-primary-abg">
    <!-- ... -->
  </div>
```

**Example 2:** use border color primary `--color-primary-abd`

```vue
<div class="border-primary-abd">
  <!-- ... -->
</div>
```

**Example 3:** use background color with opacity `--color-primary-abd`

```vue
<div class="bg-primary-abd/20">
  <!-- ..... -->
</div>
```
